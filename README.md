\# Sports Betting Analytics Platform



Personal AI-powered sports betting analytics system with real-time odds monitoring and predictive modeling.



\## Features



\- ✅ Real-time odds fetching from 40+ bookmakers

\- ✅ Market consensus probability calculation

\- ✅ Soft vs sharp book discrepancy detection

\- ✅ Kelly Criterion bet sizing

\- ✅ Historical backtesting framework

\- ✅ Performance tracking \& analytics
